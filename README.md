# macOS Web Simulation

This repository contains a clean and web-based simulation of the macOS desktop interface built using React and Vite. You can find the source files inside codebyharshsharma/react-based-macos-web-simulation.

## Features

* Desktop Environment: An interactive space featuring a bottom app dock and a top status bar with a live clock.

* Window System: Supports opening multiple draggable app windows that mimic native macOS styles.

* Simulated Apps: Replicates fun tools like a Command Line Interface (CLI), Notes, a Spotify player, a GitHub tracker, and a Resume PDF viewer.



## Quick Start

To run this project on your own machine, just type these commands into your terminal one by one:

1. Clone the repository: git clone https://github.com/codebyharshsharma/react-based-macos-web-simulation.git
2. Navigate to the folder: cd react-based-macos-web-simulation
3. Install the dependencies: npm install
4. Run the app locally: npm run dev

## Tech Stack

* Core: React 18 and Vite

* Styling: Custom CSS modules

* Packages: react-markdown, react-console-emulator etc...
